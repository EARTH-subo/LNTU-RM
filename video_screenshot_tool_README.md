# 视频截图工具使用说明

## 工具简介

视频截图工具是一个基于 PySide6 和 OpenCV 开发的图形化应用程序，用于从视频文件中按指定数量均匀截取图片帧。该工具支持多种视频格式，具有友好的用户界面和实时进度显示功能。

## 功能特点

- 🎬 **多格式支持**：支持 MP4、AVI、MOV、MKV、FLV、WMV 等主流视频格式
- 📸 **自定义截取**：可设置 1-10000 张图片的截取数量
- 📁 **智能输出目录**：自动创建唯一的 `pic` 文件夹，避免重名冲突
- 📊 **实时进度**：可视化进度条显示处理进度
- ⚡ **多线程处理**：后台异步处理，界面流畅不卡顿
- 🎯 **均匀分布**：自动计算帧间隔，确保截取的图片均匀覆盖整个视频

## 环境要求

- Python 3.10+
- PySide6 >= 6.4.2
- OpenCV-Python >= 4.11.0.86

## 安装依赖

确保已安装项目所需的依赖库：

```bash
pip install -r requirements.txt
```

## 使用方法

### 1. 启动工具

在项目根目录下运行：

```bash
python video_screenshot_tool.py
```

### 2. 选择视频文件

- 点击 **"选择视频"** 按钮
- 在弹出的文件选择对话框中选择要处理的视频文件
- 支持的视频格式：
  - MP4 (*.mp4)
  - AVI (*.avi)
  - MOV (*.mov)
  - MKV (*.mkv)
  - FLV (*.flv)
  - WMV (*.wmv)

### 3. 设置截取数量

- 在 **"截取图片数量"** 输入框中设置需要截取的图片数量
- 范围：1 - 10000 张
- 默认值：100 张

### 4. 开始截取

- 点击 **"开始截取"** 按钮
- 等待处理完成
- 进度条会实时显示处理进度
- 处理完成后会弹出提示框显示保存目录

## 输出说明

### 输出目录

工具会自动创建输出目录，规则如下：

- 首次运行：创建 `pic` 文件夹
- 后续运行：依次创建 `pic_1`、`pic_2`、`pic_3` 等，确保不会覆盖已有内容

### 文件命名

截取的图片文件命名格式为：

```
frame_000001.jpg
frame_000002.jpg
frame_000003.jpg
...
```

其中数字按截取顺序递增，使用 6 位数字格式，便于按文件名排序。

## 工作原理

1. **视频解析**：打开视频文件并获取总帧数
2. **帧计算**：根据设定的截取数量计算均匀的帧间隔
3. **帧提取**：按计算出的帧索引逐个提取视频帧
4. **图片保存**：将提取的帧保存为 JPG 格式图片
5. **进度更新**：实时更新处理进度显示

## 注意事项

1. **视频时长**：建议处理不超过 30 分钟的视频，过长视频可能需要较长处理时间
2. **磁盘空间**：确保有足够的磁盘空间存储截取的图片
3. **视频格式**：某些特殊编码的视频可能无法正常读取，建议使用标准编码格式
4. **处理时间**：处理时间取决于视频长度、截取数量和电脑性能

## 常见问题

### Q: 为什么提示"无法打开视频文件"？

A: 可能的原因：
- 视频文件已损坏
- 视频编码格式不被支持
- 文件路径包含特殊字符

### Q: 如何提高处理速度？

A: 可以尝试：
- 减少截取数量
- 关闭其他占用资源的程序
- 使用性能更好的电脑

### Q: 截取的图片不清晰怎么办？

A: 工具使用原始视频帧质量保存，图片清晰度取决于源视频质量。

### Q: 可以中途停止处理吗？

A: 当前版本不支持中途停止，如需停止请关闭程序窗口。

## 技术细节

### 核心模块

- **VideoCaptureThread**：视频处理线程，负责后台视频处理
- **VideoScreenshotTool**：主窗口类，负责用户界面和交互

### 关键参数

- `CAP_PROP_FRAME_COUNT`：获取视频总帧数
- `CAP_PROP_FPS`：获取视频帧率
- `CAP_PROP_POS_FRAMES`：设置当前读取帧位置
